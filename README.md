# bookstore-sql-analysis
Bookstore SQL project created using PostgreSQL. The project includes database creation, data import from CSV files, and solving business-related questions using SQL queries ranging from basic to advanced levels. Demonstrates skills in SELECT statements, filtering, sorting, aggregate functions, GROUP BY, JOINs, and subqueries.
## Tools Used
- PostgreSQL
- pgAdmin
- SQL

## Dataset
- Books
- Customers
- Orders

## Key Analysis
- Top-selling books
- Customer purchase behavior
- Revenue analysis
- Order trends

## Skills Demonstrated
- SQL Joins
- Aggregate Functions
- Subqueries
- Data Analysis
